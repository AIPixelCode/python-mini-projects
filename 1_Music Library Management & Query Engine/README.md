# Music Library Query Engine

## Overview

Music Library Query Engine is a lightweight Python library for storing music-related data and performing queries on users and albums. It provides a simple interface for managing users, albums, and retrieving statistics based on different filtering criteria such as artist, genre, age, and city.

The project demonstrates how structured data can be organized using Python dictionaries and accessed through reusable query functions.

---

## Features

* Add users with personal information and purchased albums.
* Store album information including artist, genre, and track count.
* Query the number of purchased tracks by:

  * User and artist
  * User and genre
  * Age and artist
  * Age and genre
  * City and artist
  * City and genre
* Modular and reusable function-based design.
